Режим установки

Рекомендуется запускать установщик Яндекс Браузера в режиме совместимости с Windows 7. Установка проходит без рекламы — смело нажимайте «Далее» на каждом шаге. Отмеченные опции добавят браузер в исключения брандмауэра, автозагрузку Windows и настроят ассоциацию с типами файлов для работы с веб-контентом. Единственное, что можно отключить — автозагрузку, всё остальное лучше оставить включенным для удобства использования.

Внимание: как и у любого старого ПО, использование устаревшей версии браузера может нести риски безопасности. Мы рекомендуем устанавливать актуальную 32‑битную или 64‑битную версию Яндекс Браузера с официального сайта.

Преимущества и недостатки версии 64 bit

Может использовать больше оперативной памяти, поэтому кэширование и работа с большими веб-страницами проходят стабильнее;

Работает быстрее на Windows x64, так как не требует запуска в среде совместимости для 32‑битных приложений;

Разработка отдельной версии 64 bit полностью прекращена — новых функций не будет;

Может быть несовместима с последними обновлениями Windows;

Иногда не запускается без объяснения причин.
